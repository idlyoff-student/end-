В этой директории должны находиться следующие изображения:

1. hero-bg.jpg - фоновое изображение для главной страницы (заголовок)
2. about-bg.jpg - фоновое изображение для секции "О нас"
3. cta-bg.jpg - фоновое изображение для секции "Свяжитесь с нами"

Рекомендуемые размеры изображений:
- hero-bg.jpg: 1920x1080 пикселей
- about-bg.jpg: 1600x900 пикселей
- cta-bg.jpg: 1600x600 пикселей

Для лучшего результата используйте изображения с хорошим качеством и оптимизированные для веб.
Изображения должны быть сжаты для быстрой загрузки страницы.

Возможные источники для изображений:
- Unsplash (https://unsplash.com/)
- Pexels (https://www.pexels.com/)
- Shutterstock (https://www.shutterstock.com/)

Пример тематики изображений:
- Для hero-bg.jpg: современное рабочее пространство с компьютерами
- Для about-bg.jpg: команда IT-специалистов или офисное пространство
- Для cta-bg.jpg: крупный план компьютерных комплектующих или человек, работающий за компьютером

Примечание: Если вы используете изображения из коммерческих источников, убедитесь, что у вас есть лицензия на их использование.
